# IPL Power BI Dashboard
## Project Overview
This project is an interactive IPL Data Analytics Dashboard developed using Power BI.
The dashboard provides insights into:
- IPL Match Summary
- Batsman Performance
- Bowler Performance
- Team Statistics
- Match Results
- Strike Rates
- Economy Rates
- Tournament Analysis
---
## Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
---
## Dashboard Pages
### 1. Overview Page
- Navigation panel
- IPL theme interface
### 2. IPL Summary
- Total matches
- Total teams
- Toss decisions
- Venue analysis
- Match winners

### 3. Batsman Stats
- Total runs
- Strike rates
- Tournament 4s & 6s
- Player comparisons

### 4. Bowler Stats
- Wickets analysis
- Economy rates
- Bowling averages
- Dismissal types

## Key Insights
- Mumbai Indians won the highest number of matches.
- SL Malinga is among the highest wicket takers.
- SK Raina and Virat Kohli are top run scorers.
- Field-first toss decisions were more successful.
